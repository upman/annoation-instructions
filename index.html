Instructions
The sentences below is from text chats having English and Hindi words. They need to be transformed to dictionary English/Hindi words. For each word:
  • Select 'Looks good' if the word belongs to the English dictionary. Leave the input below the list Empty.
  • Select 'Acronym' for words that are short forms of words. Example: lol, irl, etc. Type the expanded form of the acronym in the input below.
  • Select 'Short Form' for words that are shortened versions of english Words. Example: plz, y, u, ur etc. Type the dictionary form of the word in the input below.
  • Select 'Hindi' if the word is a Hindi word. Enter the spelling from Google translate as shown in the screenshot in the input below. Do not translate the word to english
  • Select 'Merge' for all words where there is an unnecessary space between words. Type the combined word in one of the inputs below.
  • Select 'Split' for words that need to be split into two or more separate words. Type the split words into the input below.
  • Select 'Typo' for words with spelling mistakes in them. Example: deleat, exersice, etc.
  • Select 'Wordplay' when extra vowels/consonants are used (heeeeelppp!) or a phonetic spelling of the word(lyf, iz, da) is used. Type the dictionary form in the input below.
  • Select 'Profanity' if the words are profane.
  • Select 'Unrecognizable or other language' if the word is from another language or if you cannot recognize it as a word.
  • Do not correct names of people, if they appear with the wrong spelling.
